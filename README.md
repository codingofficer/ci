# Cigar Inspector Theme

Minimal, fast starter for Cigar Inspector (news + directory + reviews).

## Install
1. Copy `cigar-inspector/` into `wp-content/themes/`.
2. Activate in WP Admin → Appearance → Themes.
3. Ensure CPTs/Taxonomies are registered (MU plugin) and ACF Local JSON is enabled.

## Structure
- `inc/` PHP setup, assets, queries, schema, helpers
- `template-parts/` reusable cards/components
- `assets/` CSS/JS
